VR版的水果忍者+射箭游戏

写博客的目的不是为了应付面试，原因一是想保存和展示下自己原来做的东西，二是觉得应该有个好习惯，谁不是从0到1呢，希望各位小弟写的东西的粗略

Situation

steam上有一款vr水果忍者，卖68.我就想着能不能自己实现下，把了下逻辑觉得太难。遂搞之。做完玩着玩着，有个大问题，一个手去砍，另一个手没事干！贼类。遂看到又有个射箭游戏，这样双手都解放了。

Task

基本功能都实现了。放两个图。视频太大还不知道怎么搞。

Action

水果忍者部分

其实核心代码就是切割部分。Blinded_AM_ME插件有个meshcut组件。直接利用OnCollisionEnter带入碰撞物体和位置信息即可。

cut的核心比较强大，主要是通过碰撞点的位置，针对两边去添加triangles三角网格。

生成水果就比较简单聊，协程给个初速度角度就ok

射箭游戏部分

主要就是绑定箭，和拉箭这两个函数，还有一个主意的是箭飞出的时候要改变下箭的朝向，有自由落体的感觉

Result

流畅实现运行，玩着和steam的相同。
